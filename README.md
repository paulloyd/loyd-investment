# LOYD Investment Partners — High-Fidelity Landing Page

Professional, cinematic landing page for LOYD Investment Partners. Built with React, GSAP, and Tailwind CSS.

## 🚀 Technology Stack

- **Core:** [React 18+](https://react.dev/)
- **Animation:** [GSAP (GreenSock)](https://gsap.com/) with ScrollTrigger
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide-React](https://lucide.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)

## 🛠 Features

- **Cinematic Experience:** High-performance GSAP animations and scroll-driven transitions.
- **Micro-Interactions:** Magnetic buttons and interactive feature artifacts.
- **Section Contrast:** Balanced luxury aesthetic using a curated Obsidian/Champagne palette.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports.
- **Lead Generation:** Integrated strategy session scheduling.

## 📦 Deployment

This project is configured for deployment on **Vercel** via **GitHub**.

### Local Development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
